## ADDED Requirements

### Requirement: 生活记录最简创建
系统必须支持以标题和日期作为最小必填字段创建生活记录，可选附加图片与感想。

#### Scenario: 仅使用最小字段创建记录
- **WHEN** 成员仅填写标题与日期提交记录
- **THEN** 系统必须成功保存为草稿状态

### Requirement: 草稿与发布状态流转
系统必须支持生活记录在草稿与已完成状态间按操作流转，并在详情中可追溯关联提议信息（若存在）。

#### Scenario: 草稿发布
- **WHEN** 成员在草稿记录执行完成发布操作
- **THEN** 系统必须将记录状态更新为已完成并保留关联提议信息
